# FindBlood
Web application that helps blood donation agencies and centers to manage their blood banks and reach potential donors.
## Built with:
  * ReactJs
  * MaterialUI
  * React Router
  * NodeJs
  * ExpressJs
  * MongoDB
  * Mongoose
## Tasks:
  - [x] Database Model
  - [x] REST Api
  - [x] Email service
  - [x] Auth
  - [x] UI components
  - [x] State management
  - [x] Responsive
  - [x] Routing
  - [x] Account verification
  - [x] Forgot password
